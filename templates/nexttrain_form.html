Welcome to UK Train Time Tables

Please Enter your Train Station Code

The Train Station Code is a three letter code for your choosen train station. E.g. 'RMD' = Richmond, 'ZPU' = East Putney, 'LVJ' = Liverpool James Street

If you do not know your train station id then click here to find a list of possible train station codes. Make sure to choose the first three letter code in brackets in the column with the station name.

Note: the http://www.railwaycodes.org.uk/stations/stationa.shtm website also provides a list of duplicate codes for england.
